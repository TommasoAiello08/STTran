# Results (STTran)

Prefer the **project-root** `../results/` for report artefacts. This folder mirrors the same layout if you keep outputs inside `STTran/`.

| Subfolder | Contents |
|-----------|----------|
| `plots/` | Frequency / Lorenz plots, predicate-score boxplots, graph-structure figures |
| `tables/` | R@K summaries from `scripts/eval_ag_recall_from_terminal_logs.py` |
| `qualitative/` | Scene graphs from `plots/viz_terminal_scene_graphs.py` |

Plotting and log-parsing code: **`plots/`** package.
